removeFromPolyTexture

Function

To provide a point-and-click method to remove the current facet selection from an existing Polygon Projection Texture.

Usage

  1. Select the facet components to remove to the Polygon Projection Texture.

  2. In the MultiLister, highlight the Shading Group that is assigned to the Polygon Projection.

  3. Select the poly__Proj node for the Projection Texture from which you want to remove the facets.

  4. Call the 'removeFromPolyTexture' function.
