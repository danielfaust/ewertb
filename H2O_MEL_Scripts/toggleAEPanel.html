toggleAEPanel

Function

Maya 2.0 now places the Attribute Editor as a panel replacement for the Channel Box. Normally the standard 'CTRL-a' hotkey toggles the Attribute Editor with the Channel Box. Use this command to show or hide the panel completely. The command remembers which panel was displayed before it was hidden and restores the appropriate mode.

I have this mapped to 'CTRL-A' (ie. 'CTRL-SHIFT-a').

Requirements

This script requires Maya v2.
