toggleDoubleSided

Function

Allows you to quickly toggle the 'doubleSided' attribute for an object.

Note: 'doubleSided' is an attribute of the shape node. If the selected node is not or does not have a shape (e.g. the root transform of a NURBS cube) this function will return a warning; e.g.:

// Warning: Cannot set 'doubleSided' attribute for "nurbsCube" //
