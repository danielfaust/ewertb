addToPolyTexture

Function

To provide a point-and-click method to add the current facet selection to an existing Polygon Projection Texture.

Usage

  1. Select the facet components to add to the Polygon Projection Texture. You may select facets which are already part of the projection, but it is not required.

  2. In the MultiLister, highlight the Shading Group to assign to the select facets.

  3. Select the ‘poly__Proj‘ node for the Projection Texture to which you want to assign the facets.

  4. Call the ‘addToPolyTexture’ function.
