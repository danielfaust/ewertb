polyCenterPoint

Function

Provides the "Add Center Point" functionality I grew accustomed to in SOFTIMAGE|3D.

To use: Pick one or more edges of a polygon object and call polyCenterPoint.

Note: Maya's Polygons->Subdivide function will do the same thing with Subdivisions set to 1. However, ‘polyCenterPoint’ is slightly more intelligent. Polygons->Subdivide will fail if the selected components' type is not part of the selection pick mask; ‘polyCenterPoint’ will correctly parse the current selection irregardless of the selection pick mask.
