findFilterExpand

Function

A highly elegant method of determining if the current selection type may be parsed with the ‘filterExpand’ command. Several selectionMask values are undocumented.

Usage

Pick anything (such as Lattice Points) and run this script. The output will indicate any selectionMask values that match the selection type.

Example output:

46:
ffd1Lattice.pt[0][2][1]
ffd1Lattice.pt[0][3][1]

Limitations

Only values of 0 - 256 (inclusive) are evaluated.
