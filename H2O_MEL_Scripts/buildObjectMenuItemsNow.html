buildObjectMenuItemsNow

The ‘buildObjectMenuItemsNow.mel’ script replaces the default installed as:

/usr/aw/maya1.0/scripts/startup/buildObjectMenuItemsNow.mel

WARNING! As far as I know, the original contents of this function have been static from Maya 1.0 through Maya v2.5.1. However, I have not been able to compare with Maya v3.

It modifies the popup menu that appears when you hold the right mouse button in a model view with nothing selected. The following command is added the the default of ‘Select All’:

Minimize Application

You need not replace the original script in your Maya install directory; simply place this script within your ‘MAYA_SCRIPT_PATH’, restart Maya, and it will politely perform the enhancement.
