selectFacetSG

Function

Determines the Shading Groups and File textures that shade the selected polygon facet components, and highlights the appropriate nodes in the MultiLister.

Usage

Select one or more facet components on a polygon object, then call ‘selectFacetSG’. The script determines which Shading Groups are used to shade the selected facets. These Shading Groups will be highlighted in the MultiLister.

Next, the input color connection for each Shading Group is checked and, if the color is derived from a File texture node, that File node will also be highlighted.

If the MultiLister is in Select Mode, as opposed to Highlight Mode, the appropriate nodes will be selected instead of highlighted.
