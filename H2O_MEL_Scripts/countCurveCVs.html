countCurveCVs

Function

Reports the number of CVs in all selected nurbsCurve objects.

Usage

Select any number of objects, presumably including at least one NURBS curve. Objects which are not NURBS curves are ignored.

Execute this script. The results will be displayed in the Script Editor.

Number of CVs in "curve1": 4
Number of CVs in "curve2": 5
Number of CVs in "curve3": 11
Number of CVs in "curve4": 4
