How do I toggle the Polygon statistics in the Heads-up Display?

Note: This How-To was introduced back in Maya v3.0, when the state of the Heads-Up Display would not persist when Maya was restarted. As of Maya v4.0 the commands for controlling this display changed, and the HUD now stays where you put it. The text below has been updated for Maya 6.

The "macro" command for this is:

TogglePolyCount;

Which is a run-time command mapping to:

setPolyCountVisibility(!`optionVar -q polyCountVisibility`);

The commands above toggles the state. If you want to specifically show or hide the Poly Count HUD, use the following:

// Show the Poly Count HUD
//
setPolyCountVisibility( on );

// Hide the Poly Count HUD
//
setPolyCountVisibility( off );
